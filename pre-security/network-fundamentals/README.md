# 📖 Overview

This room introduces the fundamentals of computer networking and explains how devices communicate across local and global networks.

Networking is one of the most important skills for every cybersecurity professional, especially penetration testers.

---

# 🎯 Learning Objectives

- Understand what networking is.
- Learn the OSI Model.
- Understand LAN and WAN.
- Learn how data travels through a network.
- Understand packets and frames.

---

# 💡 Key Takeaways

- Networks allow computers to communicate efficiently.
- The OSI Model provides a framework for understanding network communication.
- Understanding networking is essential before learning enumeration, scanning, and exploitation.

---

# 🚀 Why This Lab Matters

A solid understanding of networking is essential for penetration testing because every attack begins with discovering and communicating with systems over a network.
