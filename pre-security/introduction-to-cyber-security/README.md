# 📖 Overview

This room introduces the fundamental concepts of cybersecurity, including the different security domains, career paths, and the distinction between offensive and defensive security.

It provides the foundational knowledge required before learning penetration testing and ethical hacking.

- Learn the difference between Offensive Security and Defensive Security.

- Understand basic security terminology.

---

# 💡 Key Takeaways

- Cybersecurity is not only about hacking but also protecting systems and information.
- Offensive Security focuses on finding vulnerabilities before attackers do.
- Defensive Security focuses on detecting, preventing, and responding to attacks.
- Ethical hacking is performed with authorization to improve security.
