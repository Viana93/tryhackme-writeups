# 📖 Overview

This room introduces the basic components of a computer system and explains how hardware and software work together.

---

# 🎯 Learning Objectives

- Understand computer hardware.
- Learn about CPUs and memory.
- Understand operating systems.
- Learn how storage devices work.

---

# 📚 Topics Covered

- CPU
- RAM
- Storage
- Motherboard
- Operating Systems
- Input & Output Devices

---

# 🛠️ Skills Acquired

- Computer Hardware
- Operating Systems
- Memory Management
- Storage Devices

---

# 💡 Key Takeaways

A solid understanding of computer hardware and operating systems is essential before learning Linux, Windows, Active Directory and Ethical Hacking.

---

# 🚀 Why This Lab Matters

Cybersecurity professionals need to understand how computers work before learning exploitation, malware analysis, and system administration.
