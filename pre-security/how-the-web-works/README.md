# 📖 Overview

This room explains how websites work, how browsers communicate with servers, and the technologies behind web applications.

---

# 🎯 Learning Objectives

- Understand HTTP and HTTPS.
- Learn how DNS works.
- Understand web requests and responses.
- Learn how websites are delivered to users.

---

# 📚 Topics Covered

- HTTP
- HTTPS
- DNS
- Web Servers
- Browsers
- URLs

---

# 🛠️ Skills Acquired

- HTTP Requests
- DNS Resolution
- Client-Server Communication
- Web Fundamentals

---

# 💡 Key Takeaways

- Every web application depends on HTTP and DNS.
- Understanding web communication is essential before learning web exploitation.

---

# 🚀 Why This Lab Matters

Web application security is one of the most important areas of penetration testing. These concepts are fundamental before studying OWASP Top 10, Burp Suite, and Web Exploitation.
